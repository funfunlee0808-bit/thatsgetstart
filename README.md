# 品味小舖 · GitHub Pages 網站

這個專案使用 Google 匯出的評論、圖片與店家資訊 CSV 檔，打造了品味小舖的單頁網站，方便直接部署到 GitHub Pages。

## 使用方式
1. 將程式碼推送到 GitHub，並在專案設定中開啟 GitHub Pages（選擇根目錄作為來源）。
2. 發佈後，造訪 Pages 網址即可看到自動載入的評論、環境照片與店家亮點。

## 本地預覽
直接在專案根目錄啟動任意靜態伺服器，或是使用 VS Code 的 Live Server。

```bash
python -m http.server 8000
# 瀏覽 http://localhost:8000
```
